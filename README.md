# Telegram Task Bot PRO

Полнофункциональный бот для управления системой заданий с админ-панелью и ивентами.

## 📋 Функции

### Для участников:
- 🎮 Отправка 10 типов заданий
- 📊 Личный профиль со статистикой
- 🏆 Топ-10 игроков
- 📋 История выполненных заданий
- 🏅 Система значков и достижений
- 🎪 Участие в ивентах
- ✏️ Настройка никнейма

### Для администраторов:
- 👑 Полная админ-панель
- 📋 Проверка заданий
- 👥 Управление участниками
- 💰 Начисление/списание баллов
- 🏅 Выдача значков
- 🎭 Установка кастомных эмодзи
- 🎪 Создание и управление ивентами
- 📊 Подробная статистика системы

## 🚀 Установка (Windows)

1. Установите Python 3.8+ с сайта [python.org](https://python.org)
2. Скачайте файлы бота
3. Запустите `install.bat`
4. Отредактируйте файл `.env`:
   - Добавьте токен бота от @BotFather
   - Добавьте ваш Telegram ID (можно получить у @userinfobot)
5. Запустите бота через `run.bat`

## ⚙️ Настройка

### Получение токена бота:
1. Откройте Telegram и найдите @BotFather
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

### Получение своего Telegram ID:
1. Откройте Telegram и найдите @userinfobot
2. Отправьте `/start`
3. Скопируйте ваш ID
